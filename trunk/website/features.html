Features of Bactome

    Toolbox for Studying Bacteria, Archaea, and Complex Taxons
In Bactome II: Analyzing Gene List for Gene Ontology Over-Representation (2011)
  • Parser for version 1.0 and 2.0 Gene Ontology Annotation file (GAF)
  • Compares sample gene set with GAF to calculate expectation and report Gene Ontology over-representation counts for further statistical analyses (such as Chi-Square test)
In Bactome I: Python in DNA Fingerprinting (2010)
  • Calculating generation time of suspension cells with optional cell size correction equation
  • Finding primers for DNA fingerprinting using a reference genomic sequence
  • Finding fragment sizes after restriction endonuclease digestion
  • Calculates Nei and Li genetic distance from fragment sizes of digested DNA (RFLP; restriction fragment length polymorphism)
